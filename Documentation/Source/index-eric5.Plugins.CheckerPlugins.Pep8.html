eric5.Plugins.CheckerPlugins.Pep8

Package containing the PEP 8 plugin.

Modules

Pep8Checker Module implementing the PEP 8 checker.
Pep8CodeSelectionDialog Module implementing a dialog to select PEP 8 message codes.
Pep8Dialog Module implementing a dialog to show the results of the PEP 8 check.
Pep8Fixer Module implementing a class to fix certain code style issues.
Pep8NamingChecker Module implementing a checker for PEP-8 naming conventions.
Pep8StatisticsDialog Module implementing a dialog showing statistical data for the last code style checker run.
