eric5.Plugins.VcsPlugins.vcsMercurial

Package implementing the vcs interface to Mercurial

It consists of the hg class, the project helper classes and some Mercurial specific dialogs.

Packages

BookmarksExtension Package implementing the bookmarks extension support interface.
ConfigurationPage Package implementing the the Mercurial configuration page.
FetchExtension Package implementing the fetch extension support interface.
GpgExtension Package implementing the gpg extension support interface.
PurgeExtension Package implementing the purge extension support interface.
QueuesExtension Package implementing the queues extension support interface.

HgCommandDialog Module implementing the Mercurial command dialog.
HgCommitDialog Module implementing a dialog to enter the commit message.
