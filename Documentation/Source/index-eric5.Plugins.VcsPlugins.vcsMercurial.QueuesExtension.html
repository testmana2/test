eric5.Plugins.VcsPlugins.vcsMercurial.QueuesExtension

Package implementing the queues extension support interface.

Modules

HgQueuesFoldDialog Module implementing a dialog to enter data to fold patches.
HgQueuesHeaderDialog Module implementing a dialog to show the commit message of the current patch.
HgQueuesListDialog Module implementing a dialog to show a list of applied and unapplied patches.
HgQueuesNewPatchDialog Module implementing a dialog to get the data for a new patch.
HgQueuesRenamePatchDialog Module implementing a dialog to enter the data to rename a patch.
ProjectHelper Module implementing the queues extension project helper.
queues Module implementing the queues extension interface.
