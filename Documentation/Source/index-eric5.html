eric5

Package implementing the eric5 Python IDE (version 5.5).

To get more information about eric5 please see the eric web site.

Packages

Cooperation Package containing the modules dealing with cooperation.
DataViews Package containing modules for viewing various data.

E5Gui Package implementing some special GUI elements.
E5Network Package implementing some special network related objects.
